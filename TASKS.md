# TASKS.md

## Правило выполнения
Выполняй задачи строго по порядку.
Перед началом каждой задачи:
1. Кратко опиши план
2. Перечисли файлы, которые будут созданы или изменены
3. После реализации проверь, что проект собирается без ошибок

Не переходи к следующей крупной задаче без завершения текущей.

---

## ЭТАП 1. Инициализация и архитектура проекта

### Задача 1.1
Создай frontend-проект на React + TypeScript.

Требования:
- Используй Vite
- Настрой базовую структуру проекта
- Удали лишний шаблонный код
- Подготовь чистую стартовую архитектуру

Ожидаемый результат:
- Проект запускается локально
- Нет лишнего демо-кода
- Есть чистая структура папок

---

### Задача 1.2
Организуй структуру директорий.

Используй такую структуру:

src/
  app/
  pages/
  components/
  sections/
  shared/
  assets/
  styles/
  data/

Ожидаемый результат:
- Папки созданы
- Структура логична и масштабируема

---

### Задача 1.3
Настрой базовую архитектуру приложения.

Что нужно сделать:
- Настроить React Router
- Сделать главный layout
- Подготовить Header и Footer
- Подготовить базовую обертку страницы
- Настроить контейнеры и основные UI-элементы

Ожидаемый результат:
- Есть каркас сайта
- Есть маршрутизация между страницами
- Header и Footer подключены глобально
